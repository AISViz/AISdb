Tools and utilities for processing AIS data

Setup:
  ```
  python3 -m pip install --upgrade numpy pysqlite3-binary pyais shapely requests rasterio packaging selenium
  ```

Requirements:
  * Python 3
  * SQLite >= 3.35
  * NumPy
  * pyais

Optional:
  * qgis
  * Shapely
  * requests
  * selenium
  * rasterio
